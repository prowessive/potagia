#potagia

## Definición general

Potagia es un sistema diseñado para facilitar la creación de aplicaciones web modernas y escalables.

- Todos los archivos se almacenan en la base de datos.

- La definición del modelo de base de datos se puede importar y exportar a un archivo JSON.
- Si se cambia el json, se actualiza la base de datos.

- Existe una tabla para gestionar las rutas.

- Existe una tabla para gestionar los permisos de las rutas. (raw)
- Roles: Administrador, Usuario, Invitado
